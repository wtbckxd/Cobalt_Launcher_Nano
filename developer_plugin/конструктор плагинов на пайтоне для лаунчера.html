📱 Подсказки
  • В командах используйте api (экземпляр PluginAPI) – например, api.get_data_dir()
  • Хук баннера: def my_banner(banner): return banner + "\\nДоп. строка"
  • Хук информации: def my_info(lines): lines.append("Новый сервер"); return lines
  • Можно использовать только эти импорты: math, random, calendar, struct, binascii, unicodedata, sqlite3, os, sys, json, zipfile, requests, subprocess, platform, shutil, re, webbrowser, getpass, signal, threading, time, itertools, from pathlib import Path. from datetime import datetime. from typing import dist, list, optional, any, callable. from colored import fg, attr from typing import Dict, List, Optional, Any, Callable. tarfile, urlib.parse.
